# 이동수 · 조은지 모바일 청첩장

배포 주소: `https://dongsoo-eunji.github.io/wedding/`

## GitHub Pages 설정

1. GitHub에 `wedding` 저장소를 생성합니다.
2. 이 압축 파일의 내용물을 저장소 루트에 업로드합니다.
3. `Settings → Pages`에서 다음을 설정합니다.
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)

## 수정할 곳

- 문구와 기본 정보: `index.html`
- 디자인: `assets/css/style.css`
- 달력과 복사 기능: `assets/js/main.js`
- 사진: `assets/images/`

모든 경로는 상대경로로 작성되어 `/wedding/` 하위에서도 동작합니다.
